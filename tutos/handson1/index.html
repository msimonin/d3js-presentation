Hands-on 1 : DOM/SVG manipulations

Modify the source file to use D3 and get this behaviour :

When loading the page

  • a svg element is inserted in the DOM
  • 10 circles appear
  • then, after some time, each of them is transitioned to a zero-radius circle
  • then, then after some time, each of them is removed from the DOM

Note : You'll use a for loop to generate the 10 circles. It's definitely not a good practice with D3js (see data join section).

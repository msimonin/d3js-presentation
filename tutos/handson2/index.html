When loading the page :

  • An array of data is generated : e.g [{id:0, height:207}, {id:1, height:134}, ...]
  • Such datas are bound to rects element whose height is set according the height value of the data
  • Calling oneStep() regenerate a new array of data and update the existing rect on the screen.
